Il file corsi_formazione.sql contiene l'intero database,compresi tutti i trigger.
Il file query.sql contiene tutte le 24 operazioni che il sistema deve offrire all'azienda,sopra ogni usecase è presente una stringa che descrive in maniera sintetica la query sottostante.
Il file video simulazione_interfaccia_grafica.webm illustra, in breve, alcune funzionalità del database attraverso un interfaccia grafica,illustro come il sistema gestisce anche i casi di errore: non è possibile inserire due persone con lo stesso codice fiscale, o provare ad inserire un record senza inserire tutti gli attributi identificati come "not null".
Il file "documentazione-progetto-corsi-formazione-1962536.pdf" illustra nel dettaglio ogni singola parte dello sviluppo dell'applicazione,dalla specifica dei requisiti alla costruzione delle tabelle,trigger,query,etc..
La directory "webapp" contiene i file php necessari per la creazione di un interfaccia grafica per comunicare con il database,il file da eseguire è "home.php"
